# sd-projeto

Uma aplicação web front-end minimalista com HTML, CSS e JavaScript. Este repositório contém um pequeno projeto estático composto por `index.html`, `style.css`, `script.js` e a pasta `img/` para ativos visuais — ideal como ponto de partida para demos, protótipos e estudos.

## Visão geral

- **Leve:** HTML/CSS/JS puro, sem dependências externas.
- **Estrutura limpa:** arquivos fáceis de entender e estender.
- **Ótimo para aprendizado:** perfeito para praticar layout, DOM e interações.

## Recursos

- Página principal pronta (`index.html`).
- Estilos organizados em `style.css`.
- Comportamento e interações em `script.js`.
- Imagens e ativos em `img/`.

## Preview

Adicione um screenshot dentro da pasta `img/` (por exemplo `img/preview.png`) e inclua-o aqui para uma visualização rápida.

## Requisitos

- Qualquer navegador moderno (Chrome, Edge, Firefox, Safari).

## Instalação & Uso

1. Clone este repositório:

```
git clone https://github.com/OCostaPedro/sd-projeto.git
cd sd-projeto
```

2. Abra `index.html` no navegador (duplo clique ou arraste para uma janela do navegador).

Alternativamente, você pode servir o projeto com um servidor estático para desenvolvimento:

```
# Exemplo com Python 3
python -m http.server 8000
# Depois abra http://localhost:8000
```

## Estrutura do projeto

- `index.html` - Página principal
- `style.css` - Estilos do projeto
- `script.js` - Lógica e interações
- `img/` - Imagens e ativos
- `README.md` - Documentação (você está aqui)

## Desenvolvimento

- Edite `style.css` para ajustar o visual.
- Edite `script.js` para alterar comportamento e interações.
- Teste no navegador e verifique a pasta `img/` para imagens usadas.
